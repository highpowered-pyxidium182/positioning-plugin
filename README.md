## 🚀 What this is

positioning-plugin is a Claude Code plugin that helps startups and teams work through a structured positioning statement exercise.

Use it to shape a clear view of:
- who you serve
- what problem you solve
- how you are different
- why your message matters

It is designed for Windows users who want a simple way to run the tool and start the exercise without digging through setup steps.

## 💻 What you need

Before you start, make sure you have:

- a Windows PC
- an internet connection
- enough space to download the release file
- permission to run apps on your computer

If the release includes a desktop app, you can open it like a normal Windows program. If it includes a package for Claude Code, you can place it in the right folder and use it from there.

5. Run the app or setup file inside it

1. Open the download folder

4. Load or enable the plugin from your plugin list

## 🧰 How to use the plugin

Once the plugin is installed, open Claude Code and start the positioning exercise.

A typical session helps you define:

- your target user
- the pain point you address
- the result you want to create
- the reason your team is the right fit
- the words you should use in your message

Use the prompts in order. They are built to guide you step by step, so you do not need to know marketing terms before you begin.

## 📋 Typical workflow

A simple workflow looks like this:

1. start the plugin
2. answer the first prompt
3. review the next prompt
4. refine your statement as you go
5. save the final version for your team

You can use it for:
- new startups
- rebrands
- sales pages
- pitch decks
- team alignment
- founder messaging

## ✍️ What the exercise covers

The positioning exercise usually walks through these parts:

- customer segment
- main problem
- core value
- point of difference
- proof or reason to trust
- short positioning statement

This helps you turn vague ideas into a message that is easier to use in real work.

## 🧭 Example output

A finished positioning statement may look like this:

We help small teams turn messy ideas into clear positioning so they can explain what they do and why it matters.

Your own version will depend on your product, team, and audience. The plugin helps you shape that language in a clear order.

## 🔧 Troubleshooting

### The file will not open
- make sure the download finished
- try the latest release
- check that you picked the Windows file

### Windows blocks the app
- right-click the file
- choose Run as administrator if needed
- confirm the prompt if Windows asks for approval

### I cannot find the downloaded file
- open your Downloads folder
- sort by date
- look for the newest file from GitHub

### The plugin does not appear in Claude Code
- confirm the files are in the right folder
- restart Claude Code
- check whether the plugin needs to be enabled in settings

## 📁 Folder tips

To keep things simple, use one folder for the download and another folder for the extracted files.

A good setup is:
- `Downloads` for the original file
- `Documents\positioning-plugin` for the extracted content
- a notes file for your final positioning statement

This makes it easier to find the plugin again later.

## 🧠 Best results

For clear output, enter short and direct answers. It helps to:
- name your audience in plain words
- describe one main problem
- avoid broad claims
- keep your wording simple
- use the same term for your customer throughout the exercise

If your team works on product, marketing, or sales, it also helps to do the exercise together so you can agree on the same message.
